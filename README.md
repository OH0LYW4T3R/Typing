# Typing School
### Page : https://oh0lyw4t3r.github.io/Typing/

## 사용 언어
1. HTML
2. CSS
3. Vanilla JS
---
## JS Part
1. JS의 DOM 문법을 이용하여, HTML, CSS를 제어
2. JS의 이벤트 리스너를 통한 입출력 제어
3. 타자 속도 계산식 : ((현재 타수 - (백스페이스 횟수 * 2)) / 걸린시간) * 60
---
## 주요 기능
1. 타자 속도 측정
2. 언어 변경
3. 문장, 문단 변경
4. 다크, 라이트 모드
5. 결과 카운팅 변경
6. 키보드 인디케이터
